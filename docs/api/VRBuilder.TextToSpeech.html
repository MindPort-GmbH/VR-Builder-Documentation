FileTextToSpeechProvider

The disk based provider for text to speech, which is using the streaming assets folder. On the first step we check if the application has files provided on delivery. If there is no compatible file found, will download the file from the given fallback TextToSpeechProvider.

This factory creates and provides ITextToSpeechProviders. They are chosen by name, the following providers are registered by default:
