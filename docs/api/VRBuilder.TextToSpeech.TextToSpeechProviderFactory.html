    Class TextToSpeechProviderFactory

    This factory creates and provides ITextToSpeechProviders. They are chosen by name, the following providers are registered by default:

    • MicrosoftSapiTextToSpeechProvider
    • WatsonTextToSpeechProvider
    • GoogleTextToSpeechProvider
