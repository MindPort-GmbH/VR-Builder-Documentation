    Class MenuItem<T>

    The Step Inspector populates "Add Behavior" and "Add Condition" buttons' dropdown menus with implementations of this class. Use either or as the generic parameter. The Step Inspector will display it with VRBuilder.Core.Editor.UI.StepInspector.Menu.MenuItem`1.get_DisplayedName. If clicked, it will use the result of GetNewItem().
