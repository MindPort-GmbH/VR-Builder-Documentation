    Returns the names of files (including their paths) that match the specified search pattern in the specified directory relative to the Streaming Assets folder.

    Declaration

    Parameters
    Type Name Description
    System.String path

    The relative path to the Streaming Assets folder. This string is not case-sensitive.
    System.String searchPattern

    The search string to match against the names of files in path. Depending on the platform, this parameter can contain a combination of valid literal path and wildcard (* and ?) characters (see Remarks), but doesn't support regular expressions.
