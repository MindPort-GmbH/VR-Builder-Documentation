Class HighlightObjectMenuItem

Namespace
VRBuilder.Editor.UI.Behaviors
Assembly
VRBuilder.Editor.dll

The Step Inspector populates "Add Behavior" and "Add Condition" buttons' dropdown menus with implementations of this class. Use either IBehavior or ICondition as the generic parameter. The Step Inspector will display it with DisplayedName.get. If clicked, it will use the result of GetNewItem().
