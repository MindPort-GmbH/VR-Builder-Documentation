Class ProcessSceneObject

Namespace
VRBuilder.Core.SceneObjects
Assembly
VRBuilder.Core.dll

This component gives a GameObject a stable, non-replicatable Globally Unique Identifier. It can be used to reference a specific instance of an object no matter where it is.
